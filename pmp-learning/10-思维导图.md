# 思维导图

## 使用顺序

1. 先看 `mindmaps/index.html`，进入浏览版总入口
2. 再看 `mindmaps/pmp-overview.html`，建立全局框架
3. 再看 `mindmaps/pmbok6-exam-map.html`，攻克考试结构
4. 最后看 `mindmaps/sprint-review-map.html`，用于冲刺复盘

## 浏览版入口

- [思维导图总入口](mindmaps/index.html)
- [PMP 全局总图（Mermaid 渲染页）](mindmaps/pmp-overview.html)
- [第六版考试框架图（Mermaid 渲染页）](mindmaps/pmbok6-exam-map.html)
- [冲刺记忆图（Mermaid 渲染页）](mindmaps/sprint-review-map.html)

## 在线托管提示

- 当前浏览版页面适合直接托管到 GitHub Pages
- Mermaid 运行资源会随站点一起发布，不依赖第三方 CDN
- 若你把整个项目推到 GitHub，可使用仓库里的 Pages 工作流直接部署
