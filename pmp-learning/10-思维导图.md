# 思维导图

## 使用顺序

1. 先看 `mindmaps/index.html`，进入浏览版总入口
2. 再看 `mindmaps/pmp-master-map.html`，建立全局骨架
3. 再看 `mindmaps/core-knowledge-map.html`，把核心知识串起来
4. 然后看 `mindmaps/exam-pitfall-map.html`，攻克高频判断
5. 最后看 `mindmaps/memory-map.html`，用于冲刺复盘

## 浏览版入口

- [思维导图总入口](mindmaps/index.html)
- [PMP 总脑图（Mermaid 渲染页）](mindmaps/pmp-master-map.html)
- [核心知识图（Mermaid 渲染页）](mindmaps/core-knowledge-map.html)
- [高频易错图（Mermaid 渲染页）](mindmaps/exam-pitfall-map.html)
- [口诀记忆图（Mermaid 渲染页）](mindmaps/memory-map.html)

## 在线托管提示

- 当前浏览版页面适合直接托管到 GitHub Pages
- Mermaid 运行资源会随站点一起发布，不依赖第三方 CDN
- 若你把整个项目推到 GitHub，可使用仓库里的 Pages 工作流直接部署
- 当前脑图内容已经集中收口到一个主源文件，供构建脚本统一派生多张脑图页面
